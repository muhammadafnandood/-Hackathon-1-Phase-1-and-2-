#### Machine Learning Example:
```
🎮 Real-World Example: Robot Learning to Walk

How a robot learns to walk using Reinforcement Learning:
1. TRIAL: Robot tries to move leg
2. ERROR: Falls down
3. REWARD: Gets points for staying upright
4. LEARN: Adjusts motor commands
5. REPEAT: After 1000s of tries, learns to walk!
```

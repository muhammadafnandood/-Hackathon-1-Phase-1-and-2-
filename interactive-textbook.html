Introduction to ROS 2

ROS 2 (Robot Operating System 2) is a set of software libraries and tools for building robot applications. It uses a publish-subscribe model for communication between nodes. This decentralized architecture makes ROS 2 highly scalable and suitable for complex robotic systems.

🎯 Key Concept

ROS 2 uses DDS (Data Distribution Service) as its middleware protocol that allows nodes to communicate in a distributed system without a central coordinator.

What is ROS 2?

ROS 2 is the next generation of ROS, designed to address the limitations of the original system while maintaining its core philosophy of providing a flexible framework for writing robot software.

💡 Did you know? ROS 2 was created to support robots that need real-time control, improved security, and production-ready features that weren't available in ROS 1.

Core Concepts

Understanding ROS 2 requires familiarity with several key concepts that form the foundation of the system.

Nodes

A node is an executable that uses ROS 2 to communicate with other nodes. Nodes can be thought of as the building blocks of a ROS 2 system. Each node is responsible for a specific function, such as reading sensor data, controlling motors, or performing computations.

Topics

Topics are named buses over which nodes exchange messages. They use a publish-subscribe model where nodes can publish messages to a topic or subscribe to receive messages from a topic. This decouples the producer from the consumer of data.

⚠️ Important: Topics are anonymous and many-to-many. Multiple nodes can publish to the same topic, and multiple nodes can subscribe to the same topic.

Services

Services provide a request-response communication pattern. Unlike topics, services are synchronous - the client sends a request and waits for the server to respond. This is useful for operations that need to complete before continuing.

Actions

Actions are designed for long-running tasks that can be preempted. They provide feedback during execution and can be cancelled if needed. This makes them ideal for tasks like navigation or manipulation.

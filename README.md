# 📚 Physical AI & Humanoid Robotics Interactive Textbook

An advanced, interactive digital textbook for learning Physical AI and Humanoid Robotics with AI-powered features.

## ✨ Features

### 🎯 Core Features
- **Interactive Chapters** - Engaging content with visual elements
- **AI Chatbot Assistant** - Get instant help while learning
- **Urdu Translation** - Bilingual support for better understanding
- **Mermaid Diagrams** - Visual representation of complex concepts
- **Quizzes & Assessments** - Test your knowledge with MCQs
- **Code Examples** - Hands-on Python and ROS 2 examples
- **Glossary** - Technical terms explained

### 🤖 AI-Powered Content Creation
This textbook uses 5 specialized AI agents for content generation:

1. **Content Writer** - Generates full chapter content from outlines
2. **Diagram Generator** - Creates Mermaid.js diagrams automatically
3. **Quiz Generator** - Creates multiple-choice questions
4. **Code Explainer** - Provides line-by-line code explanations
5. **Glossary Builder** - Extracts and defines technical terms

### 🎨 User Experience
- Clean, modern interface
- Responsive design
- Easy navigation between chapters
- Search functionality
- Bookmarking capability
- Progress tracking

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge)
- Internet connection for AI features

### Installation

1. Clone the repository:
```bash
git clone https://github.com/muhammadafnandood/-Hackathon-1-Phase-1-and-2-.git
cd -Hackathon-1-Phase-1-and-2-
```

2. Open the main textbook file:
```bash
# On Windows
start interactive-textbook.html

# On Linux/Mac
xdg-open interactive-textbook.html
# or
open interactive-textbook.html
```

3. Or simply double-click `interactive-textbook.html` in your file explorer

## 📖 Usage Guide

### Reading Chapters
1. Navigate through chapters using the sidebar
2. Click on sections to expand/collapse
3. Use the search bar to find specific topics

### AI Chatbot
1. Click the "Ask AI" button in any chapter
2. Type your question in natural language
3. Get instant, context-aware answers

### Urdu Translation
1. Toggle the Urdu translation button
2. View content in both English and Urdu
3. Switch between languages as needed

### Quizzes
1. Complete a chapter to unlock the quiz
2. Answer multiple-choice questions
3. Get instant feedback with explanations
